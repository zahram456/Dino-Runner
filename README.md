# Dino Runner

A browser-based Dino Runner game built with HTML, CSS, and JavaScript.

## Overview
Dino Runner is a lightweight endless runner game inspired by the classic offline dinosaur game. Jump over obstacles, survive as long as possible, and try to beat your high score.

## Features
- Smooth jump and obstacle movement
- Score tracking
- Increasing difficulty over time
- Game over and restart flow
- Responsive layout for desktop and mobile browsers

## Controls
- `Space` or `Arrow Up`: Jump
- `R`: Restart (if game over)

On mobile, use the on-screen interaction if available or tap the game area depending on your implementation.

## Project Structure
- `index.html`: Game layout and markup
- `style.css`: Styling and animations
- `script.js`: Core game logic

## Run Locally
1. Clone the repository:
   `git clone https://github.com/zahram456/Dino-Runner.git`
2. Open the project folder.
3. Launch `index.html` in your browser.

No build tools or dependencies are required.
